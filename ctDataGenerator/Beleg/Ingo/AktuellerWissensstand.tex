\section{Der aktuelle Wissensstand}\label{DerAktuelleWissensstand}
Noch nicht vorhandene Kenntnisse über die Erstellung von Ausführunsplänen werden hauptsächlich aus Onlineressourcen bezogen. Primärliteratur zur gewählten Datenbank
ist unter www.postgresql.org zu finden. Unter dieser Adresse ist das gewählte
DBMS dokumentiert. Auf dieser Seite wird eine Einführung in PostgreSQL gegeben.
Es gibt Installationsanleitungen, eine umfassende Dokumentation von PostgreSQL und Werkzeuge die ein effizienteres Arbeiten mit PostgreSQL ermöglichen.
